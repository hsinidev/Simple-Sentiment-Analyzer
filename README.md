## ğŸ“– About The Project

In the age of big data, understanding *how* people feel is just as important as *what* they say. The **AI Sentiment Analyzer** is a modern web application designed to democratize access to advanced Natural Language Processing (NLP).

By integrating directly with **Google's Gemini 2.5 Flash model**, this tool offers enterprise-grade sentiment analysis (Positive, Negative, Neutral) with high-precision confidence scoring. It features a stunning, galaxy-themed UI designed for immersion and ease of use.

## ğŸš€ Getting Started

To run this project locally, follow these steps:

### Prerequisites

*   Node.js (v16 or higher)
*   A Google AI Studio API Key ([Get one here](https://aistudio.google.com/))

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/hsinidev/sentiment-analyzer.git
    cd sentiment-analyzer
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm start
    ```
